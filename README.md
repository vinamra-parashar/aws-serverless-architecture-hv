# AWS Serverless Architecture Assignments

## Overview

This repository contains solutions for four AWS Serverless Architecture assignments implemented using AWS Lambda, Amazon EC2, Amazon S3, Amazon EBS, IAM, EventBridge, and Boto3 (AWS SDK for Python).

The assignments demonstrate how serverless computing can be used to automate infrastructure management, storage operations, security monitoring, and backup processes.
